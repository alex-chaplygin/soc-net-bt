# Социальная сеть БТ
## Класс Сообщение
- - -
Представляет собой сообщение в чате или публикация на стене
### Поля
### `Пользователь пользователь`
Пользователь, кому принадлежит сообщение или публикация
### `string текст`
Текст сообщения
### `List<Image> изображения`
Изображения в сообщении или публикации
### Методы
### `Сообщение(Пользователь п)`
Создает сообщение от пользователя с ссылкой "п"
### `Пользователь ПолучитьПользователя()`
Возвращает ссылку на пользователя, от кого сообщение
### `void ЗадатьТекст(string т)`
Устанавливает текст в сообщении
### `string ПолучитьТекст()`
Возвращает текст сообщения
### `ДобавитьИзображение(Image и)`
Добавляет изображение с ссылкой "и" к сообщению
### `УдалитьИзображение(int номер)`
Удаляет изображение под номером "номер" из сообщения
### `List<Image> ПолучитьИзображения()`
Возвращает ссылку на список всех изображений сообщения

## Класс Чат
- - -
Служит для хранения данных личной переписки между двумя пользователями
### Поля
### `Пользователь пользователь1`
Первый пользователь
### `Пользователь пользователь2`
Второй пользователь
### `List<Сообщение> сообщения`
Список сообщений чата
### Методы
### `Чат(Пользователь пользователь1, Пользователь пользователь2)`
Создает новый чат (личную переписку) между двумя пользователями
### `Пользователь ПолучитьПользователя1()`
Возвращает ссылку на первого пользователя
### `Пользователь ПолучитьПользователя2()`
Возвращает ссылку на второго пользователя
### `void ДобавитьСообщение(Сообщение сообщение)`
Добавляет в чат сообщение по ссылке
### `List<Сообщение> ПолучитьСообщения()`
Возвращает ссылку на список всех сообщений чата

## Класс Стена
- - -
Хранит публикации (сообщения)
### Поля
### `List<Сообщение> публикации`
Список всех публикаций
### Методы
### `Стена()`
Создает новую стену
### `void Опубликовать(Сообщение публикация)`
Добавляет на стену новую публикацию
### `void Удалить (int номер)`
Удаляет публикацию под номером "номер"
### `void Редактировать(int номер, Сообщение публикация)`
Заменяет публикацию под номером "номер" на публикацию по ссылке "публикация"
### `List<Сообщение> ПолучитьПубликации()`
Возвращает ссылку на список всех публикаций на стене

## Класс Пользователь
- - -
Хранит ресурсы, принадлежащие пользователю, такие как личные сообщения и публикации на стене
### Поля
### `string имя`
Имя пользователя для авторизации
### `string пароль`
Пароль пользователя
### `Стена стена`
Стена публикаций пользователя
### `List<Чат> чаты`
Список из чатов (личных переписок) пользователя
### Методы
### `Пользователь(string имя, string пароль)`
Создает нового пользователя с заданным именем и паролем
### `string ПолучитьИмя()`
Возвращает имя пользователя
### `string ПолучитьПароль()`
Возвращает пароль пользователя
### `Стена ПолучитьСтену()`
Возвращает ссылку на стену пользователя
### `List<Чат> ПолучитьЧаты()`
Возвращает ссылку на список чатов пользователя
### `void Опубликовать(string текст)`
Добавляет публикацию с текстом "текст" на стену пользователя
### `void ПрикрепитьИзображение(Image и)`
Добавляет изображение по ссылке "и" к последней публикации на стене
### `void УдалитьПубликацию()`
Удаляет последнюю публикацию на стене
### `void НаписатьСообщение(string кому, string текст)`
Добавляет сообщение с текстом "текст" в личную переписку с пользователем, имя которого задано в параметре "кому"

## Класс СоциальнаяСеть
- - -
Хранит все ресурсы социальной сети: пользователей, личные переписки, управляет всей сетью
### Поля
### `List<Пользователь> пользователи`
Список всех пользователей, зарегистрированных в социальной сети
### `List<Чат> чаты`
Список всех личных переписок и сообщений от пользователей социальной сети
### Методы
### `СоциальнаяСеть()`
Создает новую социальную сеть
### `Пользователь Регистрация(string имя, string пароль)`
Регистрирует нового пользователя с заданным именем и паролем в социальной сети. Если уже существет пользователь с таким именем, то возвращает пустую ссылку. При успешной регистрации возвращает ссылку на нового пользователя
### `void УдалитьПользователя(string имя)`
Удаляет пользователя из социальной сети
### `Пользователь Авторизация(string имя, string пароль)`
Выполняет проверку заданных имени пользователя и пароля. В случае успешной авторизации возвращает ссылку на пользователя, иначе возвращает пустую ссылку
### `Чат НовыйЧат(string имя1, string имя2)`
Создает новую личную переписку между двумя пользователями с заданными именами и возвращает ссылку на чат. Если пользователи однаковы или один из пользователей не существует, то возвращает null
